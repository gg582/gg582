## 3. 과거 프로젝트 상세 (취업/학습 중심)

### IncuSpeed
- **개요**: Incus 컨테이너 관리 도구 (KivyMD 기반 Python GUI + Go 보안 데몬 백엔드)
- **배경**: 대구대학교 MoNet 연구실 내 테스트 환경 가속화 프로토타입
- **주요 내용**: 복잡한 가상화 대신 시스템 컨테이너(Incus)를 이용한 재현 가능한 일련의 샌드박스 프로비저닝 자동화 구현.

### DevOpsPlayground
- **개요**: Ansible, Helm, Shell 스크립트 기반 Kubernetes 재현 가능 인프라 구성 도구 모음
- **배경**: 대구대학교 MoNet 연구실 클라우드 인프라 실험용 템플릿

### RemoteCarFromMonet
- **개요**: Kubernetes를 결합한 분산 제어 기반 원격 차량 제어 시스템
- **스택**: C, Go, Kubernetes

### RIOTOSMiniCarImplementation
- **개요**: 곤충학(개미) 주행 알고리즘을 모사한 RIOT OS 디바이스 드라이버 프로젝트
- **스택**: C, RTOS, GPIO 주변장치 제어

### Raspberry Pi Raspbot
- **개요**: I2C 및 핀 인터럽트를 이용한 초음파/모터 디바이스 드라이버
- **스택**: C, Linux Device Driver

### AvianRaptorNet
- **개요**: 조류 시각 경로에서 영감을 얻은 PyTorch 기반 초소형 이미지 분류기
- **스택**: Python, PyTorch

### ChoiCrypt
- **개요**: 최석정의 지수귀문도 수리적 셔플 알고리즘을 결합한 AES-256 변형 실험 암호화 도구
- **스택**: C
